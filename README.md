
# Development

```
poetry install
```

## Start server

```
make dev-build
make dev-up
```

It runs docker-compose with all requirement services at http://localhost:8006


## Fill DB 

```
make dev-fill-db
```

## Run checks

```
make mypy
```

### Твоя задача:

1. Поднять проект
2. Наполнить БД (При успехе в консоли ты увидишь нужную тебе информацию)
3. Создать депозит (Я скину curl запрос на создание депозита)
4. Подключиться к базе данных либо через консоль, либо через любой удобный инструмент (например, DBeaver)
5. Вывести статус, uuid, id_in_payment_system и currency депозита с помощью raw sql запроса
6. Сделать коллбэк запрос (Я скину curl запрос коллбэка)
7. Убедиться, что депозит перешел в успешный статус
8. Починить все mypy предупреждения

На протяжении всего пути у тебя будут возникать разного рода ошибки. Тебе нужно их исправлять в процессе прохождения.
